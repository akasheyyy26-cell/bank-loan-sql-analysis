# 🏦 Bank Loan Portfolio Analysis — MySQL

Analyzing a bank's loan portfolio using MySQL to uncover 
risk patterns, defaulters, and officer performance.

---

## 📌 Business Problem
A bank wants to:
- Identify high-risk loan customers
- Track repayment defaulters
- Measure loan officer performance
- Understand portfolio distribution by loan type

---

## 🗄️ Database Schema (5 Tables)

| Table | Description |
|-------|-------------|
| customers | Customer demographics & income |
| loans | Loan details & status |
| repayments | Monthly payment records |
| loan_officers | Branch officer details |
| loan_applications | Application decisions |

---

## 📊 Key Insights
- 80% of loans are Active — healthy portfolio
- 3 customers have Late/Missed payments — flagged as defaulters
- Chennai branch officer handles highest loan volume
- Home loans dominate portfolio value at 60%+

---

## 🛠️ SQL Concepts Covered

| Query | Concept |
|-------|---------|
| Q1 | INNER JOIN — 2 tables |
| Q2 | LEFT JOIN — find missing records |
| Q3 | 3-Table JOIN |
| Q4 | 4-Table JOIN |
| Q5 | JOIN + WHERE + GROUP BY |
| Q6 | LEFT JOIN + CASE WHEN |
| Q7 | Window Function — RANK() OVER PARTITION BY |
| Q8 | Subquery — correlated filter |
